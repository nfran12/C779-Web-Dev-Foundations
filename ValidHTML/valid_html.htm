Below are some best practices for HTML coding:

  • Close the Tags!
  • Use meaningful title tags.
  • Use lowercase. You don't have to, but do it anyways.
  • Always add an alt - required for code to validate.
  • Use the right elements in the right place. Don't div everything.
  • Validate code - W3C or developer's toolbar
